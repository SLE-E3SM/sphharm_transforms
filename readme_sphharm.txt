sphharm_transform is a program to convert delta sea level spherical harmonics to a gauss legendre grid using the spectral to spatial routines used in Holly's sea level code.

Change filenames within sphharm_transform.f90
(May need netcdf condo env to compile)

to compile: gfortran sphharm_transform.f90 -o sphharm_transform.exe
to run: ./sphharm_transform.exe
